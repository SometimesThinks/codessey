# 실행 결과

이 문서는 프로그램의 모든 구현 기능(7가지 메뉴 + 힌트 + 랜덤 + 예외 처리)을 단계별로 테스트한 결과입니다.

---

## [기능 1] 메인 메뉴 및 입력 검증 테스트

데이터가 비어있거나 잘못된 범위의 숫자를 입력했을 때, 재입력을 유도하는 철저한 검증 프로세스를 확인합니다.

```text
==============================
파이썬 기초 문법 퀴즈
==============================
1. 퀴즈 풀기
2. 퀴즈 추가
3. 퀴즈 삭제
4. 퀴즈 목록
5. 최고 점수 확인
6. 퀴즈 풀이 기록
7. 종료
==============================
메뉴를 선택하세요: 2

==============================
퀴즈를 추가합니다!
==============================
문제: (입력 없이 엔터)
입력이 비어있습니다. 다시 입력해주세요.
문제: 파이썬의 창시자는?
1. 귀도 반 로섬
2. 제임스 고슬링
3. 빌 게이츠
4. 리누스 토발즈
정답: 1
퀴즈가 추가되었습니다!
```

---

## [기능 2] 전체 퀴즈 목록 조회

추가된 문제를 포함하여 현재 저장된 모든 퀴즈의 정보를 한눈에 확인할 수 있습니다.

```text
메뉴를 선택하세요: 4
==============================
퀴즈 목록
==============================
1. 파이썬의 창시자는?
   1. 귀도 반 로섬
   2. 제임스 고슬링
   3. 빌 게이츠
   4. 리누스 토발즈
   정답: 1

2. 리스트의 길이를 구하는 함수는?
   1. size()
   2. len()
   3. count()
   4. length()
   정답: 2
...
==============================
```

---

## [기능 3] 퀴즈 플레이 및 힌트 시스템

사용자가 0번을 입력하면 오답을 하나 제거해 주는 힌트 시스템과 점수 차등 부여(힌트 사용 시 5점, 미사용 시 10점)가 정확히 작동합니다.

```text
메뉴를 선택하세요: 1
총 2개의 퀴즈 중 풀고 싶은 퀴즈 수를 입력하세요: 1

==============================
퀴즈를 시작합니다!
==============================
파이썬의 창시자는?
1. 귀도 반 로섬
2. 제임스 고슬링
3. 빌 게이츠
4. 리누스 토발즈
정답(힌트 보기는 0번 입력): 0
힌트: 3번은 정답이 아닙니다!
정답(힌트 보기는 0번 입력): 1
정답입니다! (힌트 사용: +5점)

==============================
총 점수: 5점
==============================
```

---

## [기능 4] 최고 점수 확인

현재까지 진행한 모든 세션 중 가장 높은 점수를 실시간으로 저장하고 출력합니다.

```text
메뉴를 선택하세요: 5
==============================
최고 점수 : 45
==============================
```

---

## [기능 5] 상세 풀이 이력 실시간 확인

각 플레이 세션의 날짜, 최종 점수, 풀이 문항 수를 기록하여 리스트로 보여줍니다.

```text
메뉴를 선택하세요: 6
==============================
퀴즈 풀이 기록
==============================
1. 2026-04-06 15:52:15 - 45점 (5문제)
2. 2026-04-06 16:05:10 - 5점 (1문제) <-- 방금 플레이한 기록 실시간 반영
==============================
```

---

## [기능 6] 퀴즈 삭제 및 안전 종료

선택한 번호의 퀴즈를 삭제하고, `finally` 구문을 통해 어떤 상황에서도 모든 데이터를 `state.json`에 영구 보존하며 정상 종료합니다.

```text
메뉴를 선택하세요: 3
삭제할 퀴즈 번호를 입력하세요: 1
퀴즈가 삭제되었습니다! (추가했던 테스트 문제 삭제)

메뉴를 선택하세요: 7
(프로그램이 안전하게 종료되며 데이터가 저장됩니다)
```

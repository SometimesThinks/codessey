### ubuntu 컨테이너 실행 및 진입 후 명령어 수행

```bash

$ docker run -it ubuntu /bin/bash
```

### Docker 컨테이너 종료/유지 차이 관찰 기록

1. attach
   - **관찰:** 실행 중인 컨테이너의 메인 프로세스에 직접 연결해서 제어함
   - **결과:** 터미널에서 `exit` 명령을 입력하면 메인 프로세스가 종료되어 컨테이너도 함께 중지됨

2. exec
   - **관찰:** 실행 중인 컨테이너 내부에 새로운 프로세스 생성 후 추가로 실행함
   - **결과:** 터미널에서 `exit` 명령을 입력해도 추가된 프로세스만 종료될 뿐, 메인 프로세스는 유지되어 **컨테이너가 계속 실행(Up) 상태**를 유지함

**결론:** 컨테이너의 가동 상태를 유지하면서 내부 작업을 수행하려면 `exec` 명령을 사용하는 것이 안전

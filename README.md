# AI / SW 개발 워크페이스 구축

## 프로젝트 개요

터미널 기본 조작과 Docker를 활용한 개발 워크페이스 구축에 익숙해지는 걸 목표로 한다.

## 실행 환경

- OS: macOS 13.7.8
- Shell: zsh 5.9
- Docker: 28.4.0
- Git: 2.43.0

## 프로젝트 구조

```text

├── Dockerfile          # Nginx 기반 커스텀 이미지 빌드 설정

├── app/
│   └── index.html      # 정적 웹 페이지 콘텐츠
├── assets/             # 검증 증거 이미지 파일 (VS Code 연동, 브라우저 접속 등)
│   ├── git-vscode 연동 사진.png

    ├── 9.Docker 볼륨 영속성 검증.md
```

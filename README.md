# Git과 함께하는 Python 첫 발자국

## 프로젝트 개요

Git과 Python을 활용한 퀴즈 게임 프로젝트입니다.

## 실행 환경

- Python 3.12.1

## 프로젝트 구조

```text
.
├── main.py          # 프로그램의 진입점 및 메인 메뉴 루프 제어
├── models.py        # 퀴즈(Quiz) 및 게임 로직(QuizGame) 클래스 정의
├── utils.py         # 문자, 숫자 입력 검증 함수
├── constants.py     # 기본 퀴즈 데이터 및 파일 경로 등 고정값 정의
├── docs/            # 프로젝트 문서 및 퀴즈 결과 보고서
│   ├── test_results.md # 전체 기능 테스트 내역 및 실행 결과
│   └── git_logs.md     # 모든 기능 완성까지의 git log 기록
├── state.json       # [생성됨] 사용자 퀴즈 데이터 및 최고 점수 저장소
├── .gitignore       # Git 추적 제외 파일 설정
└── README.md        # 프로젝트 정보 및 개발 진행 상황
```

## 데이터 파일 설명(state.json)

```text
state.json
├── best_score (int) # 최고 달성 점수
├── history (list) # 퀴즈 풀이 이력 리스트
│ └── [record] (dict) # 개별 풀이 정보 {date, score, total_questions}
└── quizzes (list) # 사용자 추가 퀴즈 리스트
└── [quiz] (dict) # 개별 퀴즈 {question, choices, answer}
```

## 퀴즈 주제와 선정 이유

프로젝트의 퀴즈 주제는 **"파이썬 기초 문법 (Python Basics Quiz)"**입니다.
프로젝트의 개발을 파이썬으로 진행하기 때문에 파이썬 문법이 익숙치 않은 분들을 위해 이 주제를 선정했습니다.

## 실행 방법

프로젝트 루트에서 `python3 main.py` 명령어를 실행하여 프로그램을 시작합니다.

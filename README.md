### 공통 입력

- [ ] 숫자 입력 예외 처리
  - [ ] 입력 앞뒤 공백 제거
  - [ ] 숫자 변환 실패시, 안내 메시지 출력 후 재입력 흐름
  - [ ] 허용 범위 밖 숫자 입력시, 안내 메시지 출력 후 재입력 흐름
  - [ ] 빈 입력인 경우, 안내 메시지 출력 후 재입력 흐름
- [ ] 프로그램 실행 중 비정상 종료하지 않도록 처리
  - [ ] Ctrl+C(KeyboardInterrupt) 입력시, 비정상 종료하지 않도록 처리
    - [ ] 안내 메시지 출력하고 가능한 범위에서 저장 후 프로그램 종료
  - [ ] 스트림 종료(EOFError)시, 비정상 종료하지 않도록 처리
    - [ ] 안내 메시지 출력하고 가능한 범위에서 저장 후 프로그램 종료
- [ ] 데이터 파일 없거나 손상된 경우에도 프로그램 실행 가능
  - [ ] 파일 없는 경우, 기본 퀴즈 데이터 사용
  - [ ] 파일 손상된 경우, 기본 퀴즈 데이터로 복구(또는 초기화)
